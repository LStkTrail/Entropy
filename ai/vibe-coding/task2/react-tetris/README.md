# 俄罗斯方块 (React + JS)

基于 React + JavaScript 的经典俄罗斯方块小游戏，使用 Vite 构建。

## 运行

```bash
npm install
npm run dev
```

浏览器打开终端显示的本地地址（如 `http://localhost:5173`）即可游玩。

## 操作

| 按键 | 功能 |
|------|------|
| ↑ | 旋转方块 |
| ← → | 左右移动 |
| ↓ | 加速下落 |
| 空格 | 硬降（瞬间落底） |

## 说明

- 消除一行得 100 × 当前等级 分；等级随分数提升，下落速度加快。
- 分数达到 500、1000、1500… 时等级+1，最高 10 级。
